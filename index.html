What is bootstrap? Lets find out!

Bootstrap is the most popular HTML, CSS, and JS framework for developing responsive, mobile first projects on the web.

https://getbootstrap.com/

Bootstrap is a great and free open-source front-end web framework for use in websites and web applications. It uses HTML and CSS and only concerns itself in front-end development! It is so popular that even on Github.com it is the second most "starred" project!

It is compatible with all popular browsers including:

  • Google Chrome
  • Mozilla Firefox
  • Internet Explorer
  • Opera
  • Safari

Did you know?

Bootstrap used to be called Twitter Blueprint!

Time line of Bootstrap

  • August 19, 2011 - Initial Release of Bootstrap
  • January 31, 2012 - Bootstrap 2 released
  • 19 August 2013 - Bootstrap 3 released
  • October 29, 2014 - Bootstrap 4 announced to be in development
  • August 19, 2015 - First alpha version of Bootstrap 4 released
  • November 24, 2015 - Latest stable version of Bootstrap (3.36)
  • December 8, 2015 - Latest preview release version of Bootstrap (4.0.0-alpha.2)
